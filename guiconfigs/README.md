# guiconfigs

Configs for GUI apps that live outside `~/` and can't be managed with stow. Applied manually.

## vscode/

VS Code / VSCodium `settings.json`. Copy to the editor's user settings directory:

| | macOS | Linux |
|-|-------|-------|
| VS Code | `~/Library/Application Support/Code/User/` | `~/.config/Code/User/` |
| VSCodium | `~/Library/Application Support/VSCodium/User/` | `~/.config/VSCodium/User/` |
